# short-url
> [URL shortener](https://short-url-nilkesede.vercel.app)

### Usage
```
POST https://short-url-nilkesede.vercel.app
{
    "url": "https://www.google.com/"
}

RESPONSE
{
    "newUrl": "https://short-url-nilkesede.vercel.app/Z6XPGgxnd7"
}
```
